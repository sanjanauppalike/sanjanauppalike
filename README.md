## 🧠 About Me

I'm a graduate student at the **University of Illinois Chicago** (MS in Computer Science, **4.0 GPA**), graduating May 2026. My work lives at the intersection of **LLMs, multi-agent systems, and large-scale data infrastructure** — turning research-grade ideas into things that actually run in production.

🔭 **Currently** — Software Engineer Intern at **Qualcomm**, where I built a KPI-driven RAG pipeline (LangChain + LangGraph) and a text-to-SQL system on AWS that cut compute costs by 8% (five-figure annual savings).

🌱 **Previously** — Senior Software Engineer at **Samsung Research India**, where I shipped Bixby NLU models, productionized retrieval-augmented analytics over Apache Solr + LLMs, and pushed Solr query latency down by 35–45%.

🔬 **Research** — Graduate Research Assistant at **UIC's mHealth Lab**, training BiLSTM + U-Net models to annotate ECG waveforms at **97% accuracy** and architecting a multi-agent platform for chronic disease monitoring.

🏆 **Recently won** — 🥇 1st place at **Microsoft HackWithChicago 2025** for a real-time data platform built on Pathway + LLMs.

> "Build things that work, then make them work at scale."
